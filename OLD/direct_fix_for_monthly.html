🔧 直接修正月度報表表格固定問題

問題

月度報表的日期表頭在手機滾動時會滑掉，CSS position: sticky 不可靠。

解決方案

使用 JavaScript 動態固定表頭和左欄，100% 可靠。

實施步驟

步驟 1：在 monthly_report.php 中添加 CSS

在現有的 CSS 樣式中添加以下規則：

/* ========== JavaScript 表格固定 ========== */ /* 這些樣式會被 JavaScript 動態應用 */ .monthly-table.js-fixed thead { position: sticky !important; top: 0 !important; z-index: 100 !important; background: #f8f9fa !important; } .monthly-table.js-fixed tbody td:first-child { position: sticky !important; left: 0 !important; z-index: 90 !important; background: #f8f9fa !important; box-shadow: 2px 0 5px rgba(0,0,0,0.1) !important; } /* 手機優化 */ @media (max-width: 768px) { .monthly-table.js-fixed thead { top: 0 !important; } .monthly-table.js-fixed tbody td:first-child { left: 0 !important; } }

步驟 2：在 monthly_report.php 中添加 JavaScript

在 </body> 標籤前添加以下 JavaScript 程式碼：

<script> // 月度報表表格固定解決方案 document.addEventListener('DOMContentLoaded', function() { const tableContainer = document.querySelector('.table-container'); const monthlyTable = document.querySelector('.monthly-table'); if (!tableContainer || !monthlyTable) return; // 添加 JS 固定類別 monthlyTable.classList.add('js-fixed'); const thead = monthlyTable.querySelector('thead'); const firstCells = monthlyTable.querySelectorAll('tbody td:first-child'); if (!thead || firstCells.length === 0) return; // 監聽滾動事件 let isScrolling = false; tableContainer.addEventListener('scroll', function() { if (!isScrolling) { isScrolling = true; // 使用 requestAnimationFrame 優化性能 requestAnimationFrame(() => { const scrollLeft = this.scrollLeft; const scrollTop = this.scrollTop; // 動態調整表頭位置 thead.style.transform = `translateY(${scrollTop}px)`; // 動態調整左欄位置 firstCells.forEach(cell => { cell.style.transform = `translateX(${scrollLeft}px)`; }); isScrolling = false; }); } }); // 初始調整（觸發一次滾動以啟用固定） setTimeout(() => { tableContainer.scrollLeft = 1; tableContainer.scrollTop = 1; setTimeout(() => { tableContainer.scrollLeft = 0; tableContainer.scrollTop = 0; }, 50); }, 500); console.log('✅ 月度報表表格固定已啟用'); }); </script>
