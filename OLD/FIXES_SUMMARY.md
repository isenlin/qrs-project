## 🔧 技術細節

### 權限模型
```php
// 管理員：所有店櫃
if ($user['role'] === 'admin') {
    $user_stores = $stores;
}

// 督導：負責區域的店櫃  
elseif ($user['role'] === 'supervisor') {
    // 檢查督導是否負責此店櫃
    if (in_array($store['code'], $user['stores'])) {
        $user_stores[] = $store;
    }
}

// 業務/店櫃：自己負責的店櫃
else {
    if (in_array($store['code'], $user['stores'])) {
        $user_stores[] = $store;
    }
}
```
